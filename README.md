# GoodGoodStudy
好好学习，天天向上！这是一个定时自动截屏的软件。原因，受疫情影响，孩子们要上网课，而我们家长要上班，那么如何能知道孩子用电脑是在上课还是在干什么呢？通过这个软件，可以定时截屏记录，并可以按要求截多少张图片后发送到指定的 E-Mail 地址，这样，就算家长在上班，也能实时了解孩子上网课的动态了。
